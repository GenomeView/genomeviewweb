# Tutorials

This page provides a tutorials. 

## Getting started with NGS data

1.    Check out the load data page to learn how to load data and the [list of supported data formats](PrepareLoadData.md).
1.    Load the reference genome, typically a FASTA file. For improved speed, you may consider [indexing these files](Fasta.md#indexing)
1.    Load a BAM file. This files needs to be sorted and indexed. [Full BAM preparation instructions](PrepareShortReadAlignments.md)
1.    [Create and load a coverage plot](PrepareCoveragePlot.md). Coverage plots give you a summary view of the sequencing data and are in many cases sufficient
